# 🍼 HiDaddy - 프론트엔드

마음을 전하는 아빠의 첫 걸음 **HiDaddy**의 프론트엔드 레포지토리입니다. 

## 📱 주요 기능 구성

| 기능 | 상세 화면/기능 |
|------|----------------|
| ✅ 주차별 정보 | - 주차별 육아 정보 제공 |
| ✅ 챗봇 | - 육아 관련 Q&A<br>- GPT 기반 응답 |
| ✅ 미션 수행 | - 미션 목록 / 상세 / 수행 |
| ✅ 감정일기 | - 일기 작성 / 수정 / 삭제 |
| ✅ 커뮤니티 | - 글 작성 / 댓글, 좋아요, 삭제 |
| ✅ 하트 교환소 | - 하트로 육아 용품 구매 |

## 🧩 기술 스택
